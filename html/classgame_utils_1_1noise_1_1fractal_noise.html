Detailed Description

A class for generating fractal noise patterns in 2d using simplex noise.

The user can make calls to both fractal and simplex fiunctions, incase they only want a single leyer of noise. The simplex noise function was implemented from explanations and examples found at http://weber.itn.liu.se/~stegu/simplexnoise/simplexnoise.pdf
