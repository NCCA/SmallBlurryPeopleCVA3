\subsection{Detailed Description}
A class for generating fractal noise patterns in 2d using simplex noise. 

\hyperlink{namespace_the}{The} user can make calls to both fractal and simplex fiunctions, incase they only want a single leyer of noise. \hyperlink{namespace_the}{The} simplex noise function was implemented from explanations and examples found at \href{http://weber.itn.liu.se/~stegu/simplexnoise/simplexnoise.pdf}{\tt http\+://weber.\+itn.\+liu.\+se/$\sim$stegu/simplexnoise/simplexnoise.\+pdf}
